Aircraft database (registration & type)

Plain dump1090-fa and the SBS feed don't include registration or aircraft type per plane — only readsb/tar1090 do. VliegmasjienPRO fills the gap automatically: each new aircraft is looked up once and cached here, so the list, type filters and statistics fill in over time. To load a whole database at once (e.g. if your receiver has no internet), import a CSV/NDJSON below.

—

Communication frequencies (map layer)

Toggle the 📻 Freqs layer on the map to see airport radio frequencies (tower, ground, approach, ATIS…) near your view. Data is the public-domain OurAirports dataset — download it once below (a few MB), then it's served locally and offline.
